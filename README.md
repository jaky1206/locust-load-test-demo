# Locust Load Test Demo

## About
This project is a class assignment designed to demonstrate the use of Locust for load testing. It utilizes API built with .NET 6.

## Getting Started
To get started with this project, clone the repository and follow the instructions below.

### Prerequisites
- .NET 6 SDK
- Python 3.x
- Locust

### Installation
1. Clone the repository:
git clone https://github.com/jaky1206/locust-load-test-demo.git

2. Navigate to the project directory:

cd locust-load-test-demo

3. Install the required Python packages:

pip install -r requirements.txt


## Usage
To run the load test, execute the following command:

locust -f locustfile.py
